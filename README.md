## MaYoS Client v1.0 初回リリース
   
   ### 使い方
   1. `MaYos.dll` と `MaYoLauncher.exe` を同じフォルダに配置
   2. Minecraft (統合版) を起動
   3. `MaYoLauncher.exe` を実行 → "Inject MaYoS Client" をクリック
   
   ### 機能
   - ゲーミングカラーUI
   - Spotify HUD
   - Discord Rich Presence
   - 専用ランチャー
SCR - https://github.com/Dev-MaY0/MaYoS-Client-SCR
